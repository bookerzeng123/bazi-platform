# 🗺️ 部署操作路线图（按顺序执行）

> 目标：把你的八字网站上线，面向全球用户，500元搞定。

---

## 阶段一：今天完成（准备工作）

### 1.1 注册 Groq（5分钟，免费）

打开浏览器访问：https://console.groq.com/

1. 点击 "Sign in with Google"（用 Google 账号登录）
2. 创建一个组织（随便填名字）
3. 进入左侧菜单 → "API Keys"
4. 点击 "Create API Key"
5. 复制生成的 Key，格式类似：`gsk_xxxxxxxx`
6. 把这个 Key 保存到一个安全的地方

> 注意：这个 Key 只能看一次，关了页面就找不到了。

---

### 1.2 注册 Vercel（5分钟，免费）

打开：https://vercel.com

1. 点击 "Sign Up"
2. 用 GitHub 账号登录（如果没有 GitHub 先注册一个）
3. 跳过引导页面

---

### 1.3 注册 GitHub（10分钟，免费）

打开：https://github.com

1. 点击 "Sign up"
2. 注册一个账号（用户名记下来，后面要用）
3. 验证邮箱

---

### 1.4 购买域名（10分钟，~¥70）

推荐在 **Cloudflare Registrar** 购买：
1. 打开 https://dash.cloudflare.com/
2. 注册账号
3. 左侧菜单 → "Registrar"
4. 搜索你想要的域名（如 `astromind.ai`）
5. 加入购物车 → 结算（约 $10/年，可以用支付宝）

或者在 **Namecheap** 购买（支持支付宝）：
1. 打开 https://www.namecheap.com/
2. 搜索域名 → 加入购物车 → 结账

---

## 阶段二：本周完成（代码上传）

### 2.1 创建 GitHub 仓库

1. 打开 https://github.com
2. 点击右上角绿色按钮 "New repository"
3. 填写：
   - Repository name: `bazi-platform`
   - Description: `八字命理大师 - AI命盘解读`
   - 选择 Private（私人仓库）
   - ✅ 勾选 "Add a README file"
4. 点击 "Create repository"

---

### 2.2 上传代码

在 GitHub 仓库页面：
1. 点击 "uploading an existing file"
2. 打开文件夹 `C:\Users\Administrator\bazi-platform\`
3. 全选所有文件（Ctrl+A）
4. 拖拽到上传区域
5. 底部填写 Commit message：`Initial commit`
6. 点击 "Commit changes"

---

### 2.3 连接 Vercel

1. 打开 https://vercel.com
2. 点击右上角 "+" → "Add New..." → "Project"
3. 选择 "Import Git Repository"
4. 在列表中找到 `bazi-platform`，点击 "Import"
5. Framework Preset 自动检测到 Next.js，不需要改
6. 找到 "Environment Variables" 部分，点击 "Add"
7. 添加一个环境变量：
   - Name（变量名）: `GROQ_API_KEY`
   - Value（值）: 你在 1.1 步复制的 Key
8. 点击 "Deploy"

等待 2-3 分钟，出现 ✅ 绿勾代表成功！

Vercel 会给你一个临时网址，例如：
`https://bazi-platform-xxxx.vercel.app`

打开这个网址，你的八字网站就已经上线了！

## 阶段三：绑定自己的域名

### 3.1 在 Vercel 设置域名

1. 进入 Vercel 的 Project → Settings → Domains
2. 输入你的域名（如 `astromind.ai`）
3. 点击 "Add"
4. Vercel 会显示两行 DNS 记录

### 3.2 在域名服务商配置 DNS

以 Cloudflare 为例：

1. 打开 https://dash.cloudflare.com/
2. 进入你的域名 → DNS → Records
3. 点击 "Add record"，添加：

**记录1：**
- Type: `A`
- Name: `@`
- IPv4 address: `76.76.21.21`
- Proxy status: `DNS only`（小云朵点灭）

**记录2：**
- Type: `CNAME`
- Name: `www`
- Target: `cname.vercel-dns.com`
- Proxy status: `DNS only`

保存后等待 5-30 分钟（全球 DNS 生效）。

现在打开你的域名（如 `astromind.ai`），你的网站就上线了！

## 阶段四：配置收款

### 4.1 申请 PayPal.Me

1. 登录 https://www.paypal.com/
2. 进入"我的PayPal" → "收款"
3. 点击"获取PayPal.Me链接"
4. 设置你的用户名（如 `astromind`）
5. 你的链接就是：`https://paypal.me/astromind`

### 4.2 更新网站收款链接

1. 打开 GitHub 仓库
2. 进入 `app/consult/page.tsx`
3. 找到这一行：
   ```tsx
   href="https://paypal.me/yourpaypalid"
   ```
4. 把 `yourpaypalid` 改成你的真实 PayPal.Me 用户名
5. 提交修改（Commit changes）
6. Vercel 会自动重新部署，2分钟后生效

## 🆘 如果遇到问题

| 问题 | 解决方法 |
|------|---------|
| Vercel 部署失败 | 看红色报错信息，常见是环境变量 Key 填错了 |
| 域名打不开 | DNS 生效需要5-30分钟，等一等 |
| 排盘结果奇怪 | 检查输入的出生年月日时是否正确 |
| AI 不回答 | 检查 GROQ_API_KEY 是否正确配置在 Vercel 环境变量里 |
| 想加更多功能 | 参考 README.md 里的扩展说明 |
